# Dimwit
> The name comes from the knuckleheadedness of believing one person can be an entire software lab by themselves.

[manifesto](https://dimwit.me/manifesto.html) | [dimwit.me](https://dimwit.me/) | [deepanshkhurana [at] outlook [dot] com](mailto:deepanshkhurana@outlook.com)

----

Hi, I am [@DeepanshKhurana](https://github.com/DeepanshKhurana), the actual Dimwit behind this insane idea. The goal of Dimwit is simple:

> To build software I want to use myself in whatever time I find after work, on weekends, and in between. Open or closed source, free or paid, useful or dumb. It doesn’t matter. Life is too short not to build fun things. Then share that software with others who might find it useful. I expect to earn $0.00 doing this. Most projects will be free. Some things are simply worth building. If you’d like to nudge me along, I won’t complain.
